


# Surf Report Chrome Extension

## Install
```
nvm use // If you're using nvm. Otherwise ensure you're using the NodeJS version specified in the `.nvmrc file
npm install
```

## Develop
```
npm start
```

## Build
```
npm run build
```
The unpacked extension files can be found in `./dist`
